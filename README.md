# Cold Outreach Pipeline Research — B2B SaaS
## What this is
A curated research repository of 10 high-signal practitioners in B2B cold outreach,
built to support a playbook on what actually works in 2025.
## How content was collected
- LinkedIn posts: manually collected from each expert's profile
- YouTube transcripts: downloaded via yt-dlp, cleaned with Python
## Repository structure
- /research/sources.md — master list with annotations
- /research/linkedin-posts/ — posts per expert
- /research/youtube-transcripts/ — transcripts per expert
- /research/other/ — newsletters, additional materials
